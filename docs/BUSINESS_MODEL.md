## 💵 PRICING STRATEGY

### Pricing Philosophy

**Value-based pricing, mitte cost-based**

```
Customer value:
- Konsultant: €10,000-50,000 (ühekordne)
- NIS2 Abimees: €49/kuu = €588/aasta

Customer saves: €9,400+ (1. aastal)

Our pricing: 5-10% of alternative (consultant)
→ Easy decision!
```

### Pricing Psychology

**1. Anchoring:**
```
Landing page:
"Konsultandid maksavad €10,000+
                vs
         NIS2 Abimees €49/kuu"
         
→ €49 tundub odav!
```

**2. Decoy Effect:**
```
Pricing page:

FREE          STARTER         PROFESSIONAL
€0            €49/kuu         €149/kuu
                ⬆️ MOST POPULAR

→ STARTER tundub "just right"
```

**3. Price Tiers (Good-Better-Best):**
```
Good: FREE (trial)
Better: STARTER (sweet spot)
Best: PROFESSIONAL (aspirational)

→ 70% customers choose STARTER
→ 20% upgrade to PRO
→ 10% stay FREE
```

**4. Annual Discount (Commitment):**
```
Monthly: €49/kuu
Annual: €490/aasta (-17%)

→ "Save 2 months!"
→ Better cash flow for us
→ Lower churn
```

---

### Pricing Experiments (A/B Testing)

**MVP Launch (Month 1-3):**
```
Current: €49/kuu

Test variants:
A. €39/kuu (lower barrier)
B. €49/kuu (baseline)
C. €59/kuu (premium positioning)

Hypothesis: €49 on sweet spot (mid-market)
Measure: Conversion rate, MRR
```

**Promotional Pricing:**
```
Launch discount:
- First 100 customers: 50% off first 3 months
- "€24.50/kuu instead of €49"

Benefits:
- Creates urgency
- Lowers barrier
- Builds testimonials

Risk:
- Customers expect discount to continue
- Lower LTV

Mitigation:
- Clear communication (limited time)
- Grandfather clause (keep price if stay)
```

```
→ Net: +€200 MRR

Negative churn = healthy growth!
```

### Variable Costs (Per Customer)

**AI (Claude API):**
```
Hinnangud per customer/kuu:
- Onboarding: 10k tokens = €0.05
- Enesehindamine: 30k tokens = €0.50
- Dokumentide genereerimine: 30k tokens × 3 = €1.50
- Igakuine vestlus: 20k tokens = €1.00
──────────────────────────────
TOTAL: ~€3.00/kuu per customer

Optimization:
- Caching (repeated questions)
- Cheaper model for simple tasks (Haiku)
- Context window optimization
→ Target: €2.50/kuu
```

**Storage:**
```
Supabase Storage: €0.50/kuu per customer
(Dokumentid, tõendid, logid)
```

**Payment Processing:**
```
Stripe: 1.4% + €0.25 per transaction
Monthly: €49 × 1.4% + €0.25 = €0.94/kuu
```

**Support (amortized):**
```
1 support person handles 100 customers
Salary: €2500/kuu
Cost per customer: €25/kuu

But:
- 80% self-service (AI, docs)
- Average: €5/kuu per customer
```

**TOTAL VARIABLE COSTS:**
```
€3.00 (AI) + €0.50 (storage) + €0.94 (payment) + €5.00 (support) = €9.44/kuu

Gross profit: €49 - €9.44 = €39.56 (81% margin) ✅
```

## ⚠️ RISK MITIGATION

### Business Risks

**Risk 1: Slow Customer Acquisition**
```
Risk: Can't get to 50 customers in 6 months
Impact: Can't reach break-even, burn money

Mitigation:
- Pilot customers before launch (5-10)
- Referral program (viral growth)
- Partner channel (IT consultants)
- Paid ads (€500/month test budget)

Early warning: <5 customers by Month 2
```

**Risk 2: High Churn**
```
Risk: Customers leave after 1-2 months
Impact: LTV drops, unit economics break

Mitigation:
- Onboarding excellence (activate value)
- Monthly reports (continuous value)
- Proactive support (before they churn)
- Annual plans (commitment)

Early warning: >3% monthly churn
```

**Risk 3: Low Conversion (FREE → STARTER)**
```
Risk: Users don't upgrade from FREE
Impact: No revenue, server costs only

Mitigation:
- Clear value differentiation
- Limit FREE tier (20 messages/month)
- Upgrade prompts in UI
- Trial expiration emails

Early warning: <5% conversion rate
```

---

### Financial Risks

**Risk 4: AI Costs Explode**
```
Risk: Claude API costs >>€3/customer/kuu
Impact: Gross margin drops, unprofitable

Mitigation:
- Caching (repeated queries)
- Rate limiting (abuse prevention)
- Cheaper models (Haiku for simple tasks)
- Monitor usage per customer

Early warning: AI costs >€5/customer/kuu
```

**Risk 5: Can't Raise Prices**
```
Risk: Market won't accept price increases
Impact: Limited revenue growth

Mitigation:
- Grandfather old customers (keep current price)
- New customers pay new price
- Justify with features/value
- Gradual increases (10-20%)

Early warning: Pushback on €49 price
```

**Risk 6: Running Out of Money**
```
Risk: Burn rate too high, no revenue yet
Impact: Shutdown

Mitigation:
- Bootstrap (no/low salaries)
- Pre-sales (pilot customers pay upfront)
- Small angel round (€50k-100k)
- Side income (consulting)

Early warning: <3 months runway
```
